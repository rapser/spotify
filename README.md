# Spotify Clone

Es una aplicacion clone de Spotify que consumo el API Developer de spotify.

### Tecnología

Para el presente proyecto se utilizó lo siguiente

* UIKit Programático
* Kingfisher
* UITabBarController
* MVVM
* Swift 5
* Xcode 15.0
